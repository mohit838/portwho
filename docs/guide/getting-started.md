# Getting Started

## Clone

```bash
git clone https://github.com/mohit838/portwho.git
cd portwho
```

## Install for current user

```bash
./install.sh
```

Default install path: `~/.local/bin/portwho`

## Install system-wide

```bash
sudo ./install.sh --system
```

System path: `/usr/local/bin/portwho`
